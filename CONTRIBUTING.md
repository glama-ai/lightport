# Contributing

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev        # Start dev server with hot reload
```

## Testing

```bash
pnpm test       # Run all tests
```

## Linting & Formatting

```bash
pnpm lint       # Run linter
pnpm format     # Format code
```

## Building

```bash
pnpm build      # Build for production
```
